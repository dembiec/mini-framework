Creating routing

Routes can be configured in the routing/http.php file using two functions get() and post(). Which take two parameters address and redirection.

routing/http.php
                    use Route\Router;
                    $router = new Router();

                    $router->get('uri', 'Controller\Class::method');
                    $router->post('uri', 'Controller\Class::method');

Parameter declaration

declaring the parameter consists in entering {?} into the address.

routing/http.php
                    use Route\Router;
                    $router = new Router();

                    $router->get('uri/{?}', 'Controller\Class::method');
                    $router->post('uri/{?}', 'Controller\Class::method');

                    // example
                    $router->get('foo/{?}/bar', 'Controller\Foo::bar');
