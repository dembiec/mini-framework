Reading configuration

To read the configuration use the Utility\Config class. The returned configuration is an array. Configuration files can be created in the config/ directory

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\Config;
                    Class Foo
                    {
                        $config = Config::read('app');
                        echo $config[index];
                        // Your code   
                    }
