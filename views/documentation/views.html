Displaying view

Framework uses. The Utility\View class has been created. Call it and pass its parameter with the view name. You can create your own views in the views/ folder according to the twig documentation

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\View;
                    Class Foo
                    {
                        $twig = new View();
                        $twig->twigRender($viewName); 
                    }

Disabling cache

By default, cache is enabled. To disable it, call the twigCache() function with the FALSE parameter.

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\View;
                    Class Foo
                    {
                        $twig = new View();
                        $twig->twigCache(FALSE);
                        $twig->twigRender($viewName); 
                    }

Clearing the cache

To clear the cache, call the twigClearCache() function with the TRUE parameter.

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\View;
                    Class Foo
                    {
                        $twig = new View();
                        $twig->twigClearCache(TRUE);
                        $twig->twigRender($viewName); 
                    }

Passing own function to view

the transfer of own functionality takes place in the utilities/viewextension.php file. In the getFunctions() method.

utilities/viewextension.php
                    public function getFunctions()
                    {
                        return [
                            new TwigFunction('functionName', function(string $parameter = NULL) {
                                return Class::method($parameter); 
                            })
                        ];
                    }
