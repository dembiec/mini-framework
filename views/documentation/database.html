Performing database queries

To perform a database query, use the Utility\Database class. Calling the query() method on it and passing a query to it.

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\Database;
                    Class Foo
                    {
                        $database = new Database();
                        $database->query("SELECT * FROM `bar`");
                    }

Receiving data for a SELECT query

After executing the query, call the fetchAll() method.

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\Database;
                    Class Foo
                    {
                        $database = new Database();
                        $result = $database->query("SELECT * FROM `bar`");
                        $result->fetchAll();
                    }
