Creating a controller

The controller is created in the controllers/ directory in it with a class with the same name as the file in which it is located. What you can see in the example

controllers/foo.php
                    namespace Controller\Foo;
                    Class Foo
                    {
                        // Your code   
                    }

Receiving parameters passed in the address

The parameter that we declared in routing can be obtained by declaring the method variable. Please note that the parameter we receive is an array.
