Allowing logged in access

Use the Utility\Authorization class and call the forLogged() method.

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\Authorization;
                    Class Foo
                    {
                        Authorization::forLogged();
                        // Your code   
                    }

Prohibited access for logged

Use the Utility\Authorization class and call the forNotLogged() method.

controllers/foo.php
                    namespace Controller\Foo;
                    use Utility\Authorization;
                    Class Foo
                    {
                        Authorization::forNotLogged();
                        // Your code   
                    }
